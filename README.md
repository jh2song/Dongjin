# 소프트웨어 환경
- Visual Studio 2022
- .Net Core 3.1
- WPF
- SQLite

# 하드웨어
- FHD(1920 x 1080) 모니터가 권장됨 (그러나 대부분 해상도에서 문제없음을 확인함)
- 프로그램이 초기에 실행이 되지 않으면 .Net Core 3.1을 설치 후 실행해보세요. ([링크](https://dotnet.microsoft.com/en-us/download/dotnet/3.1))
